# 0x00. Hello, Dart

Introduction to the Dart language: basic syntax, variables, data types,
control flow and the `main()` entry point.

## Requirements

- Ubuntu 20.04 LTS
- Dart 3.x.x
- All files end with a new line

## Files

| File | Description |
|------|-------------|
| `0-hello_holberton.dart` | Prints `Hello Holberton!` followed by a new line |

## Usage

```bash
dart 0-hello_holberton.dart
```
